# MediMind

MediMind es una app móvil para registrar tratamientos, organizar recordatorios de
dosis y dar seguimiento básico al inventario de medicamentos.

La app está pensada como un asistente amable de recordatorios. No recomienda
dosis, no diagnostica y no sustituye indicaciones médicas.

## Estado actual

- Expo / React Native.
- Firebase Authentication para cuentas por correo.
- SMS real en APK instalada con Firebase Phone Auth.
- Firestore para estadísticas básicas y datos de usuario.
- Recordatorios locales con notificaciones.
- Desbloqueo local con huella/biometría cuando ya existe una sesión guardada.
- Panel admin preparado para el usuario configurado del desarrollador.
